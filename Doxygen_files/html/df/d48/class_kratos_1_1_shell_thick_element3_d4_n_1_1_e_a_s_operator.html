Detailed Description

template<ShellKinematics TKinematics>
class Kratos::ShellThickElement3D4N< TKinematics >::EASOperator

EASOperator.

This class performs some operations and stores some data to compute the membrane contribution of the stiffness matrix using the Enhanced Assumed Strain formulation.

References:

  • J.C.Simo,M.S.Rifai, "A class of mixed assumed strain methods and the method of incompatible modes", International Journal for Numerical Methods in Eng., vol. 29, 1595-1638, 1990
