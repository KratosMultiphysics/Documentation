Detailed Description

template<class TSparseSpace, class TDenseSpace>
class Kratos::ResidualBasedAdjointSteadyScheme< TSparseSpace, TDenseSpace >

A scheme for steady adjoint equations.

Solves the adjoint equations:

\[ \partial_{\mathbf{w}}\mathbf{r}^T \lambda = -\partial_{\mathbf{w}}J^{T} \]

\(\lambda\) is returned by Element::GetFirstDerivativesVector. \(\partial_{\mathbf{w}}\mathbf{r}^T\) is returned by Element::CalculateFirstDerivativesLHS. \(\partial_{\mathbf{w}}J^{T}\) is returned by ResponseFunction::CalculateFirstDerivativesGradient.

This function is designed to be called in the builder and solver to introduce the selected time integration scheme.

It "asks" the matrix needed to the element and performs the operations needed to introduce the selected time integration scheme. This function calculates at the same time the contribution to the LHS and to the RHS of the system
