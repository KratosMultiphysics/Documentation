Detailed Description

template<class TDataType, class TReturnType = std::vector<TDataType>>
class Kratos::FilteredAccumReduction< TDataType, TReturnType >

A derived template class from AccumReduction that only accumulates values based on a filtering condition.

Template Parameters
TDataTypeType of the data to be accumulated.
TReturnTypeType of the container used to store the data, defaults to std::vector<TDataType>.
