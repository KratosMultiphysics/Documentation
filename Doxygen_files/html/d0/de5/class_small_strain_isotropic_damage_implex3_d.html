Detailed Description

Damage with hardening constitutive law in 3D, using Implex integration scheme (see J Oliver et al, 2008, An implicit/explicit integration scheme to increase computability of non-linear material and contact/friction problems)

This material law is defined by the parameters:

  • YOUNG_MODULUS
  • POISSON_RATIO
  • HARDEDNING_CURVE: Type of hardening model: 0 exponential, 1 multilinear
  • STRESS_LIMITS: list of stress values in which the corresponding hardening parameter is valid
  • HARDENING_PARAMETERS: List of hardening modules (max three branches considered)
