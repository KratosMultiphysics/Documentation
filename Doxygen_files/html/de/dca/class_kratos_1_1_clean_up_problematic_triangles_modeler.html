Detailed Description

This class is responsible for cleaning up problematic geometries in a mesh, specifically null area geometries.

The class identifies and removes degenerated triangles where two nodes are in the same position. It ensures that the mesh remains water-tight by reconnecting the sides and renumbering the nodes and entities.

Note
This is a solution to clean up problematic geometries in the mesh that for example can be found in OBJ files.

This method clean up the problematic geometries (null area geometries) in the mesh.

For the moment it will be assumed that the problematic geometries are triangles with two nodes in the same position. This means several this:

  • The triangle is degenerated, one of the nodes is redundant and should be cleaned up.
  • The triangle must be removed as well. The sides must be reconnected, so the mesh is water-tight.
  • The nodes and entities must be renumbered
    Parameters
    rThisModelPartReference to the model part to clean up into.
    rEntityTypeThe entity type to create in the model part. Can be "element", "condition" or "geometry".
    FirstNodeIdThe first node ID to create in the model part.
    FirstELementIdThe first element ID to create in the model part.
    FirstConditionIdThe first condition ID to create in the model part.
    AreaToleranceThe tolerance to consider a geometry as problematic.
