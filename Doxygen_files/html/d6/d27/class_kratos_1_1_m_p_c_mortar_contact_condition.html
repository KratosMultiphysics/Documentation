class Kratos::MPCMortarContactCondition< TDim, TNumNodes, TNumNodesMaster >

MPCMortarContactCondition.

This is a contact condition which employes the mortar method with dual lagrange multiplier in a explicit manner in order to compute the gap/slip and the weights

Author
Vicente Mataix Ferrandiz
Template Parameters
TDimThe dimension of work
TNumNodesThe number of nodes of the slave
TNumNodesMasterThe number of nodes of the master
