Finds the nearest point to the given point on a line segment (given 2 points).

It first projects the point into the line. If the projected point is inside the segment boundary it returns the projected point. If not it returns the nearest end point of the line. Implementation based on shadertoy.com/view/ttfGWl.

Template Parameters

TGeometryTypeThe type of the line. Assumes to have [] access and IsInside method
Parameters

rLinePointAThe first point of the line
rLinePointBThe second point of the line
ToleranceThe tolerance to consider the point inside the line
Returns
The nearest point to rPoint

Computes the nearest point on a triangle to a given point in 3D space.

This method calculates the nearest point on a triangle defined by three points (rTrianglePoint0, rTrianglePoint1, and rTrianglePoint2) to a given point rPoint in 3D space. If the nearest point lies within the triangle, it returns the projection. If the nearest point lies on one of the triangle's edges, it returns the nearest point on that edge. Based on https://www.shadertoy.com/view/ttfGWl

Parameters
rPointThe point from which the nearest point on the triangle is to be found.
rTrianglePoint0The first vertex of the triangle.
rTrianglePoint1The second vertex of the triangle.
rTrianglePoint2The third vertex of the triangle.
Returns
Point The nearest point on the triangle to the given point.
