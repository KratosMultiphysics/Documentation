Apply damping to the given weights.

This method applies damping to the passed rWeights. The damped weights are stored in the rDampedWeights.

Parameters
rDampedWeightsOutput damped weights.
rWeightsInput filtering weights.
IndexIndex of the current damping computation.
NumberOfNeighboursnumber of neighbours around the Index point.
