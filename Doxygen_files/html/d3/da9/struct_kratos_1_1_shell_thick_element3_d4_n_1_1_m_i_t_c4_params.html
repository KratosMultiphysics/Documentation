Detailed Description

template<ShellKinematics TKinematics>
struct Kratos::ShellThickElement3D4N< TKinematics >::MITC4Params

MITC4Params.

This class performs some operations and stores some data to compute the transverse shear contribution of the stiffness matrix using the M.I.T.C. formulation.

References:

  • Dvorkin,Bathe, "A continuum mechanics based four node shell element for general nonlinear analysis", Eng.Comput.,vol. 1, 77-88, 1984
  • Bathe, Dvorkin, "Short communication A four-node plate bending element based on Mindlin/Reissner plate theory and a Mixed Interpolation", International Journal for Numerical Methods in Eng., vol. 21, 367-383, 1985
