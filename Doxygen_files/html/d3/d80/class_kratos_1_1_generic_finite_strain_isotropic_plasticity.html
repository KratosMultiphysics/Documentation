Detailed Description

template<class TConstLawIntegratorType>
class Kratos::GenericFiniteStrainIsotropicPlasticity< TConstLawIntegratorType >

This class is the base class which define all the constitutive laws for plasticity in strain framework.

This class considers a constitutive law integrator as an intermediate utility to compute the plasticity This implementation is based on the Eulerian logarithmic strain measure (COMPUTATIONAL METHODS FOR PLASTICITY THEORY AND APPLICATIONS. EA de Souza Neto,D Perić, DRJ Owen pag. 596).

Template Parameters
TConstLawIntegratorTypeThe constitutive law integrator considered
Author
Alejandro Cornejo & Vicente Mataix Ferrandiz
