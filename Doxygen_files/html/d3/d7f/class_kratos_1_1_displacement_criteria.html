Detailed Description

template<class TSparseSpace, class TDenseSpace>
class Kratos::DisplacementCriteria< TSparseSpace, TDenseSpace >

This is a convergence criteria that considers the increment on the solution as criteria.

The reactions from the RHS are not computed in the solution

Author
Riccardo Rossi
