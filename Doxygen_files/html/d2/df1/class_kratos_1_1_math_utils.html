inlinestatic

Clamps a value between a minimum and maximum range with a scenario.

This function ensures that the given value rX lies within the specified range [rMinimum, rMaximum]. If rX is less than rMinimum, it returns rMinimum. If rX is greater than rMaximum, it returns rMaximum. Otherwise, it returns rX. It also returns a scenario indicating if the value is below the minimum, above the maximum or within the bounds.

Template Parameters
TThe type of the value and bounds, typically a numeric type.
Parameters
rXThe value to clamp.
rMinimumThe minimum bound.
rMaximumThe maximum bound.
rScenarioThe scenario where the value is clamped. It can be BELOW_MINIMUM, ABOVE_MAXIMUM or WITHIN_BOUNDS.
ToleranceThe tolerance value to consider for clamping, defaults to 0.
Returns
The clamped value.
