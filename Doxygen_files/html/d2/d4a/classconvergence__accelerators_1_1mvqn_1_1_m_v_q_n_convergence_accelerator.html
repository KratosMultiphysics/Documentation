Detailed Description

Class MVQNConvergenceAccelerator.

This class contains the implementation of the MVQN method and helper functions. Reference: A.E.J. Bogaers et al. "Quasi-Newton methods for implicit black-box FSI coupling", Computational methods in applied mechanics and engineering. 279(2014) 113-132.
