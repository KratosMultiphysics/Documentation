Reshapes the given value to given shape.

This method reshapes the given rContainer to the shape given by pShapeBegin and pShapeEnd recursively.

If this method has changed size of rContainer or any of the elements of the container, then true is returned. False is returned if no change is required in rContainer to comply with the given shape represented by pShapeBegin and pShapeEnd.
