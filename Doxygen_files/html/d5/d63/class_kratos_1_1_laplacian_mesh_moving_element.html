Detailed Description

This mesh-updating scheme solves the Laplace equation in order to update the mesh. to distribute the motion of the structure into the fluid flow domain. It can be used for a) quadrilateral, b) triangular, c) tetrahedral and d) hexaedral elements, working best for b) and c),
