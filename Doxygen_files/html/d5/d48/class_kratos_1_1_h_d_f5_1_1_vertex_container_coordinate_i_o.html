Detailed Description

IO class for writing the coordinates (and IDs) of a set of vertices

Vertex coordinates are written to an (Nx3) matrix to <prefix>/POSITION. If "write_vertex_ids" is true, Vertex} IDs are written to an additional column (making the dataset an (Nx4) matrix). IDs are necessary for line outputs with MPI runs to identify the order of points along the line.

Default parameters: { "prefix" : "/", "write_vertex_ids" : false }
