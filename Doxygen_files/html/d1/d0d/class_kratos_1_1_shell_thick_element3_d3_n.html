Detailed Description

template<ShellKinematics TKinematics>
class Kratos::ShellThickElement3D3N< TKinematics >

ShellThickElement3D3N.

This element represents a 3-node Shell element based on the Discrete Shear Gap theory (DSG) by Bletzinger. This element is formulated for small strains, but can be used in Geometrically nonlinear problems involving large displacements and rotations using a Corotational Coordinate Transformation. Material nonlinearity is handled by means of the cross section object.
