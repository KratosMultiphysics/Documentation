Detailed Description

Class IQNILSConvergenceAccelerator.

This class contains the implementation of the IQN-ILS method and helper functions. Reference: Joris Degroote, PhD thesis "Development of algorithms for the partitioned simulation of strongly coupled fluid-structure interaction problems", 84-91.
