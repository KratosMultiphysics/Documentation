Detailed Description

template<unsigned int TDim>
template<unsigned int TNumNodes, unsigned int TComponentIndex>
class Kratos::QSVMSDerivativeUtilities< TDim >::ShapeDerivative< TNumNodes, TComponentIndex >

Shape derivative computation container.

This class is used to compute derivatives of effective velocity, length and strain rate with respect shape TComponentIndex component.

Template Parameters
TNumNodesNumber of nodes in the element.
TComponentIndexComponent index of the shape variable (nodal coordinates component).
