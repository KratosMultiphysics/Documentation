Detailed Description

All the processes python should be derived from "Process".

The process facilitates the discovery of neighboring nodes in a master model part within a designated radius for each node in the slave model part. Following this, it establishes a master-slave constraint and calculates its weights using a spatial function that employs radial basis functions.

Public member variables: model – the container of the different model parts. settings – Kratos parameters containing process settings.
