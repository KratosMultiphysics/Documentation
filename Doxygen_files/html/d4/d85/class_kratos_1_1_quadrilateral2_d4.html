Detailed Description

template<class TPointType>
class Kratos::Quadrilateral2D4< TPointType >

A four node 2D quadrilateral geometry with bi-linear shape functions.

While the shape functions are only defined in 2D it is possible to define an arbitrary orientation in space. Thus it can be used for defining surfaces on 3D elements. The node ordering corresponds with: v ^ | 3--------—2

+-— --> u

| | | | 0--------—1
