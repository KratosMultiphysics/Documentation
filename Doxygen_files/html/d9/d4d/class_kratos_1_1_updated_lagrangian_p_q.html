Detailed Description

Partitioned Quadrature Large Displacement Lagrangian Element for 3D and 2D geometries. (base class)

Implements a Partitioned Quadrature Large Displacement Lagrangian definition for structural analysis. This works for arbitrary geometries in 3D and 2D (base class)
