Detailed Description

This law defines an hyperelastic material according to the Saint-Venantâ€“Kirchhoff formulation for 2D-plane stress cases.

The simplest hyperelastic material model is the Saint Venantâ€“Kirchhoff model which is just an extension of the linear elastic material model to the nonlinear regime. More info https://en.wikipedia.org/wiki/Hyperelastic_material#Saint_Venant%E2%80%93Kirchhoff_model
