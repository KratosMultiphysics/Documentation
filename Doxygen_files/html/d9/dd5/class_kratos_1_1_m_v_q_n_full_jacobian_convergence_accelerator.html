Detailed Description

template<class TSparseSpace, class TDenseSpace>
class Kratos::MVQNFullJacobianConvergenceAccelerator< TSparseSpace, TDenseSpace >

MVQN acceleration scheme MultiVectorQuasiNewton convergence accelerator from Bogaers et al. 2016.

Forward declaration of standard MVQN This is required since the include ward avoids the inclusion of the standard MVQN
