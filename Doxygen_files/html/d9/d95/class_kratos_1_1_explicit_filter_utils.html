template<class TContainerType >
ContainerExpression< TContainerType > Kratos::ExplicitFilterUtils< TContainerType >::ForwardFilterField ( const ContainerExpression< TContainerType > &  rContainerExpression) const

Filters the given input.

This method filters the given control space mesh-independent input field using the following formulae where \(\tilde{\underline{s}}\) represents the control space variables, \(\underline{s}\) represents the physical space variables, \(\mathbf{D}\) is the damping matrix and \(\mathbf{A}\) is the smoothing matrix.

\[ \Delta \underline{s} = \mathbf{D}\mathbf{A}\Delta \tilde{\underline{s}} \]

Parameters
rContainerExpressionmesh-independent update field in control space.
Returns
ContainerExpression<TContainerType> Filtered/Smoothened mesh-independent update field in physical space
