Initialize the RomDatabase object.

Args: general_rom_manager_parameters: Parameters for the ROM manager. mu_names: List of parameter names. A single list with the name of each parameter to be used (for reference) in the database, e.g. ["alpha", "permeability", "mach"]. self.mu_names are expected to match the position and the number of inputs in the lists mu_train, mu_test, mu_run database_name: Name of the SQLite database file.
